ASCII ART GENERATOR

This ASCII art generator is a versatile tool that transforms input strings into eye-catching ASCII art. It offers three distinct banner styles to suit various aesthetic preferences:

Standard: A classic ASCII art style with simple, bold characters.

Shadow: Adds depth to the text by incorporating shadow effects.

Thinkertoy: A playful, minimalist style reminiscent of early computer graphics.

Features

Converts any input string to ASCII art

Supports three unique banner styles: Standard, Shadow, and Thinkertoy

Easy-to-use interface for quick ASCII art creation

Ideal for adding visual flair to console applications, text-based UIs, or creative coding projects

Usage

Users can input their desired text and select one of the three available banner styles. The generator then processes the input and outputs the corresponding ASCII art representation.
